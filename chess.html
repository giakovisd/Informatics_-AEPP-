Ο Μύθος του Σκακιού

Σύμφωνα με το μύθο ο εφευρέτης του σκακιού κλήθηκε από τον Ινδό βασιλιά να ζητήσει όποια αμοιβή ήθελε ως ανταμοιβή για το σπουδαίο νέο παιχνίδι που παρουσίασε. Ο εφευρέτης ζήτησε να κάνουν το εξής: να τοποθετήσουν 1 κόκκο ρύζι στο πρώτο τετράγωνο της σκακιέρας, 2 στο δεύτερο, 4 κόκκους ρυζιού στο τρίτο και να διπλασιάζουν τους κόκκους ρυζιού σε κάθε επόμενο τετράγωνο (επισημαίνεται ότι μια σκακιέρα περιέχει 64 τετράγωνα). Ο βασιλιάς και οι αυλικοί του απόρησαν γιατί ο εφευρέτης ζήτησε τόσα λίγα!

Να αναπτύξετε πρόγραμμα που θα υπολογίζει σε τόνους την ποσότητα του ρυζιού που συγκεντρώνεται με την παραπάνω διαδικασία. Δίνεται πως 1 κιλό ρυζιού περιέχει 20000 κόκκους.

Η μεταβλητή που αναπαριστά τους κόκκους είναι δηλωμένη πραγματική, λόγω ακρίβειας της ΓΛΩΣΣΑΣ
Η Γεωμετρική Πρόοδος (3D)
Ταχύτητα:
ΠΡΟΓΡΑΜΜΑ Σκάκι
ΜΕΤΑΒΛΗΤΕΣ
ΠΡΑΓΜΑΤΙΚΕΣ: κοκκοι, αθρκοκ
ΑΚΕΡΑΙΕΣ: i
ΑΡΧΗ
κοκκοι <- 1.0
αθρκοκ <- 1.0
ΓΙΑ i ΑΠΟ 2 ΜΕΧΡΙ 64
κοκκοι <- κοκκοι * 2.0
αθρκοκ <- αθρκοκ + κοκκοι
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
τονοι <- (αθρκοκ / 20000.0) / 1000.0
ποσοστο <- (τονοι / 512600000.0) * 100
ετη <- τονοι / 512600000.0
αιωνες <- ετη / 100.0
