🚀 Συνεχόμενα 1 (Μέγιστο Σερί)
Το Πρόβλημα
Μας ζητείται να διαβάσουμε ακολουθία από bits (0 ή 1) μέχρι να δοθεί κάτι διαφορετικό. Στόχος είναι να βρούμε το μεγαλύτερο πλήθος ΣΥΝΕΧΟΜΕΝΩΝ 1.

Χρειαζόμαστε δύο μεταβλητές:
πλ (Τρέχον Σερί):
Αυξάνεται κατά 1 όποτε έρχεται το 1. Μηδενίζεται μόλις εμφανιστεί το 0, αφού το σερί "σπάει".

max (Ρεκόρ Σερί):
Ενημερώνεται ΜΟΝΟ όταν το πλ καταφέρει να το ξεπεράσει. Δεν μηδενίζεται ποτέ!
Επερχόμενα Δεδομένα (Ουρά):
πλ <- 0
max <- -1
ΔΙΑΒΑΣΕ bit
ΟΣΟ bit = 0 Η bit = 1 ΕΠΑΝΑΛΑΒΕ
ΑΝ bit = 1 ΤΟΤΕ
πλ <- πλ + 1
ΑΝ πλ > max ΤΟΤΕ
max <- πλ
ΤΕΛΟΣ_ΑΝ
ΑΛΛΙΩΣ
πλ <- 0
ΤΕΛΟΣ_ΑΝ
ΔΙΑΒΑΣΕ bit
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
ΑΝ max <> -1 ΤΟΤΕ
ΓΡΑΨΕ max
ΑΛΛΙΩΣ
ΓΡΑΨΕ 'Δεν υπάρχει bit 1'
