Η Εντολή Μέχρις_ότου
Η εντολή Αρχή_επανάληψης ... Μέχρις_ότου ελέγχει τη συνθήκη στο τέλος του βρόχου.
Αυτό σημαίνει ότι οι εντολές εκτελούνται τουλάχιστον μία φορά! Αν η συνθήκη γίνει Αληθής, ο βρόχος σταματάει.

Εφαρμογή:
Να γραφεί αλγόριθμος που θα διαβάζει επαναληπτικά ύψη ατόμων. Για τον τερματισμό της εισαγωγής δεδομένων ο αλγόριθμος εμφανίζει το μήνυμα "Υπάρχει επόμενο άτομο; (ΝΑΙ/ΟΧΙ)".
Αν εισαχθεί η τιμή 'ΟΧΙ', να τερματίζεται η εισαγωγή. Ο αλγόριθμος τότε εμφανίζει το πλήθος των ατόμων που πληκτρολογήθηκαν.
Αναστημόμετρο
Αλγόριθμος Καταμέτρηση_Ατόμων
  πλήθος <- 0
  Αρχή_επανάληψης
    Διάβασε ύψος
    πλήθος <- πλήθος + 1
    Εμφάνισε "Υπάρχει επόμενο; (ΝΑΙ/ΟΧΙ)"
    Διάβασε απάντηση
  Μέχρις_ότου απάντηση = 'ΟΧΙ'
  Εμφάνισε πλήθος
